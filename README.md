## 📖 这本书讲什么

把 [ai-coding-guide](https://github.com/jnMetaCode/ai-coding-guide) 仓库里的 **10 款主流 AI 编程工具教程 + 7 套通用方法论 + 多个实战脚本**，按读者画像而不是按工具，重组为 **3 卷 + 1 别册** 的渐进式读本。

工具会换代，方法论永生。所以这套书的结构是：**先教你方法，再用工具佐证**。

### 卷一 · 入门：AI 编程从零开始

> 适合：AI 编程零基础 / 只用过 Tab 补全

| 章节 | 内容 |
|------|------|
| §0 | 9 工具速查表（30 秒选工具） |
| §1 | 提示词工程（"四元素"模板） |
| §2 | 需求拆解（任务给 AI 之前先切块） |
| §3 | 上下文管理（防止对话越长越笨） |
| §4-7 | 4 款工具速通：Cursor / GitHub Copilot / Claude Code / OpenAI Codex CLI |

### 卷二 · 工作流：实战进阶

> 适合：已经会用 1-2 款工具，想把效率往上拔

| 章节 | 内容 |
|------|------|
| §1-3 | 代码审查 / 调试方法论 / 测试策略 |
| §4-7 | 多工具协作：选型 / 实战脚本 / Claude Code + Cursor / Claude Code + Copilot |
| §8-12 | 4 款主流工具的 31 个深度陷阱（症状 / 根因 / 出坑 / 预防） |
| §13-17 | 进阶工具：Aider / Gemini CLI / Windsurf / Trae / Kiro |

### 卷三 · 架构：把 AI 编程接进团队

> 适合：团队 lead / 平台工程师 / CTO

| 章节 | 内容 |
|------|------|
| §1 | 安全注意事项（先把红线讲清楚） |
| §2 | OpenClaw — 开源 AI Agent 框架 |
| §3 | 延伸学习资源（外部高质量参考） |

### 别册 · 速查手册

> 案头放一份，每天翻两眼

| 内容 |
|------|
| 9 工具横向能力矩阵 + 30 秒决策表 |
| 每款工具的核心命令 + 配置文件路径 + 快捷键 |
| 5 套推荐组合方案 |

## 🎯 适合谁读

- **新手**：完全没用过 AI 编程，想从零开始正确地学一遍
- **进阶**：已经在用 Cursor/Copilot 但效率"还能更高"
- **团队**：想把 AI 编程接进 CI、安全策略、企业治理
- **创作者**：写自媒体内容需要靠谱、有源码出处的参考

## 🔬 信息可信度

每条 CLI flag、配置字段、子命令、模型名都对照过相应仓库的源码（特别是 OpenAI Codex CLI 那章，**6 轮源码核实**）。社区博客里常见的过期信息（如 `--full-auto`、`bubblewrap` 等）已修正——任何被否决的"常识"都附了源码出处链接。

工具迭代快，每章末尾标注"信息核实截止日期"。本书 v1.0 截止 **2026-04-30**。
